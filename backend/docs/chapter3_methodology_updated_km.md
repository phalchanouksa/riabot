## бҹЈ.бҹЎ. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹғбһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң

бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ“бҹҒбҹҮбһҮбһ¶бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бҹӮбһ”бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ (Applied Research) бһҠбҹӮбһӣбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҺбҹӮбһ“бһ¶бҹҶбһҮбҹҶбһ“бһ¶бһүбһҹбһ·бһҖбҹ’бһҹбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ·бһҹбҹ’бһҹбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһёбҹЎбҹў бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹҶбһҺбһ„бһ”бһ“бҹ’бһҸбһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹ…бһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷбһўбһ„бҹ’бһӮбһҡбҹ” бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһ“бҹҒбҹҮ бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бҹӮбһ”бһ”бһҡбһ·бһҳбһ¶бһҺбһңбһ·бһҹбҹҗбһҷ (Quantitative Research) бһҮбһ¶бһҹбҹ’бһ“бһјбһӣ бһ бһҫбһҷбһҡбһҪбһҳбһ•бҹ’бһҹбҹҶбһҮбһ¶бһҳбһҪбһҷбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– (Action Research) бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ—бһ·бһңбһҢбҹ’бһҚ бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҖбҹӮбһӣбһҳбҹ’бһўбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһұбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”бһ“бһ№бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бҹ”

бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ”бһҡбһ·бһҳбһ¶бһҺбһңбһ·бһҹбҹҗбһҷ бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣ бһ“бһ·бһ„бһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһӣбҹҒбһҒбһ–бһёбһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡ NEA бһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢ feature vector бһ…бҹҶбһ“бһҪбһ“ 256 бһ’бһ¶бһҸбһ» бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҢбҹӮбһӣ TabNetбҹ” бһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһҸбһ¶бһҳ accuracy, top-3 accuracy, confusion matrix, classification report, confidence бһ“бһ·бһ„ uncertainty бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ Python бһ“бһ·бһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Machine Learning бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бҹ”

бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– (Action Research) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳбһңбһҠбҹ’бһҸбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

- бһҖбҹҶбһҺбһҸбҹӢбһ”бһүбҹ’бһ бһ¶ (Identifying the Problem): бһҹбһ·бһҹбҹ’бһҹбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһёбҹЎбҹўбһҳбһ¶бһ“бһҖбһ¶бһҡбһӣбҹҶбһ”бһ¶бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҮбҹҶбһ“бһ¶бһүбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһ“бһ№бһ„бһ…бҹҶбһҺбһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһ“бһ·бһ„бһҮбҹҶбһ“бһ¶бһүбһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“бҹ”
- бһҡбҹҖбһ”бһ…бҹҶбһ•бҹӮбһ“бһҖбһ¶бһҡ (Planning): бһҹбһ·бһҖбҹ’бһҹбһ¶бһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢ NEA бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҺбҹӮбһ“бһ¶бҹҶбһҮбҹҶбһ“бһ¶бһүбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ TabNet бһ“бһ·бһ„ Adaptive Recommendationбҹ”
- бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– (Acting): бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Web Backend/Frontend бһ”бһ„бҹ’бһҖбһҫбһҸ pipeline бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ preprocess бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҢбҹӮбһӣ бһ“бһ·бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбҹ”
- бһҹбһ„бҹ’бһҖбҹҒбһҸ бһ“бһ·бһ„бһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Observing): бһ”бҹ’бһҡбһҳбһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһ–бһёбһҖбһ¶бһҡбһ‘бһҹбҹ’бһҹбһ“бҹҚбһ‘бһ¶бһҷ бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҢбҹӮбһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҮбһ¶бһҳбһҪбһҷ real data, team-simulated data бһ“бһ·бһ„ synthetic dataбҹ”
- бһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„ (Reflecting): бһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҹбҹҶбһҺбһҪбһҡбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһЎбһҫбһ„бһңбһ·бһү бһ“бһ·бһ„бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһұбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”бһ“бһ№бһ„бһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷбһўбһ„бҹ’бһӮбһҡбҹ”

## бҹЈ.бҹў. бһ”бҹ’бһҡбһ—бҹҒбһ‘ бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бҹӮбһҖбһҮбһ¶ бҹў бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбҹҶбһҒбһ¶бһ“бҹӢ бһӮбһә бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһҳбҹ’бһ”бһ„ (Primary Data) бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһ“бҹ’бһҹбҹҶ (Secondary Data)бҹ”

### бҹЈ.бҹў.бҹЎ. бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһҳбҹ’бһ”бһ„ (Primary Data)

бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһҳбҹ’бһ”бһ„ бһӮбһәбһҮбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бҹ’бһҡбһҳбһјбһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢ бһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- бһӣбһ‘бҹ’бһ’бһ•бһӣбһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҶбҹ’бһӣбһҫбһҷ бһҠбҹӮбһӣбһ”бҹҶбһ–бҹҒбһүбһҸбһ¶бһҳбһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢ NEA (бһ‘бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҸбһ·бһҳбһ»бһҒбһҡбһ”бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ)
- бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ real assessment бһҠбҹӮбһӣбһ”бҹ’бһҡбһҳбһјбһӣбһ”бһ¶бһ“бһ–бһёбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’
- бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ team-simulated assessment бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһҖбҹ’бһҡбһ»бһҳбһҖбһ¶бһҡбһ„бһ¶бһҡ бһҸбһ¶бһҳ persona бһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ“бҹ’бһҗбҹӮбһҳбһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһҠбһӣбҹӢ dataset
- бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ”бһҹбҹӢбһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷбһўбһ„бҹ’бһӮбһҡ бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ mapping бһ“бһ·бһ„ filtering бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

### бҹЈ.бҹў.бҹў. бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһ“бҹ’бһҹбҹҶ (Secondary Data)

бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһ“бҹ’бһҹбҹҶ бһӮбһәбһҮбһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ…бһ бһҫбһҷ бһ“бһ·бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖбһҳбһҖбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң бһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- бһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ–бһё NEA (бһ‘бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҸбһ·бһҳбһ»бһҒбһҡбһ”бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ)
- бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҮбһ¶бһҹбһ¶бһ’бһ¶бһҡбһҺбҹҲбһўбҹҶбһ–бһёбһҮбҹҶбһ“бһ¶бһүбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ”бһҹбҹӢбһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷбһўбһ„бҹ’бһӮбһҡ
- бһўбһҸбҹ’бһҗбһ”бһ‘бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң бһҹбҹҖбһңбһ—бҹ… бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„ Recommendation System бһ“бһ·бһ„ Machine Learning
- бһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„ Documentation бһҡбһ”бһҹбҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҠбһјбһ…бһҮбһ¶ Django, PyTorch TabNet, Pandas, NumPy бһ“бһ·бһ„ Scikit-learn

## бҹЈ.бҹЈ. бһ‘бҹҶбһ бҹҶбһҹбһ¶бһҖбһӣбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҹбҹҶбһҺбһ¶бһҖ

бһҹбһ¶бһҖбһӣбһҹбһ·бһҖбҹ’бһҹбһ¶ (Population) бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ“бҹҒбҹҮ бһҳбһ¶бһ“ бҹў бһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ–

- бһҹбһ·бһҹбҹ’бһҹбһҗбҹ’бһ“бһ¶бһҖбҹӢбһ‘бһёбҹЎбҹў бһҠбҹӮбһӣбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһӮбҹ„бһӣбһҠбҹ…бһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’
- бһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһҹбһ·бһҖбҹ’бһҹбһ¶бһҠбҹӮбһӣбһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷбһўбһ„бҹ’бһӮбһҡбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“ бһ“бһ·бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҺбҹӮбһ“бһ¶бҹҶ

бһ…бҹҶбһ–бҹ„бҹҮбһҹбҹҶбһҺбһ¶бһҖ (Sample) бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһёбһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡ NEA бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҳбҹ’бһӣбҹӮбһ„бһ‘бҹ…бһҮбһ¶бһ‘бһҳбҹ’бһҡбһ„бҹӢ `ch1` бһ“бһ·бһ„ `ch2` бһҹбһҡбһ»бһ” 256 бһҹбҹҶбһҺбһҪбһҡбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҠбҹҶбһ”бһјбһ„ бһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ persona sample бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҹбһ·бһҹбҹ’бһҹбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ real assessment data бһ“бһ·бһ„ team-simulated data бһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһұбҹ’бһҷбһҳбҹүбһјбһҢбҹӮбһӣбһҡбҹҖбһ“бһ”бһ¶бһ“бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ…бҹ’бһҡбһҫбһ“бҹ”

бһҠбҹ„бһҷбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһ·бһҸбһ“бҹ…бһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸ бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ hybrid training approach бһҠбҹ„бһҷбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ real data бһ“бһ·бһ„ synthetic data бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҸбһ¶бһҳ logic бһ“бҹғбһ…бҹҶбһҺбһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҡбһңбһ¶бһ„бһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбҹ”

## бҹЈ.бҹӨ. бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбҹҶбһҺбһ¶бһҖ

бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҹбҹҶбһҺбһ¶бһҖбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ“бҹҒбҹҮ бһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ Judgmental Sampling бһ¬ Purposive Sampling бһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶бҹ” бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ¶бһ“бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһҡбһҺбһёбһҠбҹӮбһӣбһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷ profile бһҹбһ·бһҹбҹ’бһҹбһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶ бһҠбһјбһ…бһҮбһ¶бҹ–

- бһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бҹҶбһҺбһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһҫбһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһҳбһҪбһҷ
- бһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһҳбһҪбһҷ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҮбҹҶбһ“бһ¶бһүбһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҒбҹ’бһӣбһ¶бҹҶбһ„
- бһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһ…бҹҶбһҺбһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢ
- бһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбһҳбҹ’бһҡбһҫбһҹ major бҹў бһ¬ бҹЈ бһҮбһ·бһҸбһӮбҹ’бһ“бһ¶

бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бҹӮбһ”бһ“бҹҒбҹҮбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ’бһ¶бһ“бһ¶бһҗбһ¶ dataset бһҳбһ¶бһ“бһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮ бһ“бһ·бһ„бһўбһ¶бһ…бһҮбһҪбһҷбһұбҹ’бһҷбһҳбҹүбһјбһҢбҹӮбһӣ TabNet бһҡбҹҖбһ“бһ–бһё pattern бһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶бһ”бһ¶бһ“бһӣбҹ’бһўбһҮбһ¶бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбһҠбҹӮбһӣбҹ—бҹ”

## бҹЈ.бҹҘ. бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

### бҹЈ.бҹҘ.бҹЎ. бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- бһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢ NEA (бһ‘бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҸбһ·бһҳбһ»бһҒбһҡбһ”бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ)
- бһ‘бһҳбҹ’бһҡбһ„бҹӢбһ”бҹҶбһ–бҹҒбһү assessment бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Web
- бһҜбһҖбһҹбһ¶бһҡ CSV бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ train/retrain
- бһҜбһҖбһҹбһ¶бһҡбһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ–бһёбһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷбһўбһ„бҹ’бһӮбһҡ

### бҹЈ.бҹҘ.бҹў. бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶ бҹў бһ•бҹ’бһ“бҹӮбһҖбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ–

#### бһҖ. бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһҳбҹ’бһ”бһ„

бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бһҳбҹ’бһ”бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣбһ–бһёбһҖбһ¶бһҡбһ”бҹҶбһ–бҹҒбһүбһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡбһҡбһ”бһҹбҹӢ NEA бһҠбҹӮбһӣбһҳбһ¶бһ“ бҹў бһ•бҹ’бһ“бҹӮбһҖбһӮбһә `ch1` (Interest) бһ“бһ·бһ„ `ch2` (Skill)бҹ” бһ…бһҳбҹ’бһӣбһҫбһҷбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢ JSON бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһ”бһҳбҹ’бһӣбҹӮбһ„бһ‘бҹ…бһҮбһ¶ feature vector бһ…бҹҶбһ“бһҪбһ“ 256 бһ’бһ¶бһҸбһ»бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Machine Learningбҹ”

бһ”бһ“бҹ’бһҗбҹӮбһҳбһ–бһёбһӣбһҫбһ“бҹҒбҹҮ бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҖбҹҸбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ team-simulated assessments бһҠбҹ„бһҷбһұбҹ’бһҷбһҹбһҳбһ¶бһҮбһ·бһҖбһҖбҹ’бһҡбһ»бһҳбһ”бҹҶбһ–бҹҒбһү assessment бһҮбһ¶бһ”бһ»бһӮбҹ’бһӮбһӣбһҖбҹ’бһӣбҹӮбһ„бһ”бһ“бҹ’бһӣбҹҶ (persona) бһҒбһ»бһҹбҹ—бһӮбҹ’бһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһ”бһҡбһ·бһҳбһ¶бһҺ бһ“бһ·бһ„бһ—бһ¶бһ–бһ…бһҳбҹ’бһҡбһ»бҹҮбһ“бҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ”бҹӮбһҖбһ–бһё real data бһұбҹ’бһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбҹ”

#### бһҒ. бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһ“бҹ’бһҹбҹҶ

бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ”бһ“бҹ’бһҹбҹҶбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣбһ–бһёбһҜбһҖбһҹбһ¶бһҡбһҡбһ”бһҹбҹӢ NEA бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ”бһҹбҹӢбһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷбһўбһ„бҹ’бһӮбһҡ бһўбһҸбҹ’бһҗбһ”бһ‘бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҳбһ»бһ“бҹ— бһ“бһ·бһ„ technical documentation бһҡбһ”бһҹбҹӢ framework бһ“бһ·бһ„ library бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ”

## бҹЈ.бҹҰ. бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ“бҹҒбҹҮ бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

1. бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data Preprocessing)
   бһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһёбһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡ NEA бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҳбҹ’бһӣбҹӮбһ„бһ‘бҹ…бһҮбһ¶ feature vector 256 бһ’бһ¶бһҸбһ» бһҠбҹ„бһҷбһ”бҹҶбһ”бҹӮбһҖбһҮбһ¶ Interest 96 бһ’бһ¶бһҸбһ» бһ“бһ·бһ„ Skill 160 бһ’бһ¶бһҸбһ»бҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҳбһјбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ format бһ“бһ·бһ„ label бһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһұбҹ’бһҷбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбҹ”

2. бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ training dataset
   бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣ real data, team-simulated data бһ“бһ·бһ„ synthetic data бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ hybrid dataset бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҢбҹӮбһӣбҹ”

3. бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһјбһҢбҹӮбһӣ
   бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫ `TabNetClassifier` бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ“бһ–бһё pattern бһҡбһңбһ¶бһ„бһ…бҹҶбһҺбһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚ бһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһӮбҹ„бһӣбһҠбҹ…бҹ” бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҮбһ¶бһҳбһҪбһҷ training set бһ“бһ·бһ„ validation set бһ бһҫбһҷбһҳбһ¶бһ“ early stopping бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„ overfittingбҹ”

4. бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҢбҹӮбһӣ
   бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҢбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһңбһ¶бһҹбҹӢбһңбҹӮбһ„бһҠбҹ„бһҷ accuracy, top-3 accuracy, confusion matrix бһ“бһ·бһ„ classification reportбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ adaptive recommendation бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҖбҹҸбһ”бһ¶бһ“бһңбһ·бһ—бһ¶бһӮ confidence бһ“бһ·бһ„ uncertainty бһ•бһ„бһҠбҹӮбһҡбҹ”

5. бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’
   бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҸбһ¶бһҳ SDLC бһ…бһ¶бһ”бҹӢбһ–бһё planning, analysis, design, implementation бһ“бһ·бһ„ testing бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бҹ”

## бҹЈ.бҹ§. бһҖбҹ’бһҡбһҳбһҹбһёбһӣбһ’бһҳбҹҢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң

бһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҠбҹ„бһҷбһӮбҹ„бһҡбһ–бһҸбһ¶бһҳбһӮбҹ„бһӣбһҖбһ¶бһҡбһҺбҹҚбһҹбһёбһӣбһ’бһҳбҹҢбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҷбҹүбһ¶бһ„бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҳбһ¶бһ“бһҖбһ¶бһҡбһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбҹ”

### бҹЈ.бҹ§.бҹЎ. бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһҹбҹ’бһҡбһ”бһ…бҹ’бһ”бһ¶бһ”бҹӢ

бһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮ бһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бһ–бһҳбһҖбһ–бһё NEA (бһ‘бһёбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһҮбһ¶бһҸбһ·бһҳбһ»бһҒбһҡбһ”бһҡ бһ“бһ·бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡ) бһ бһҫбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҷбһҖбһҳбһҖбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бҹ” бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҳбһ»бһҒбһҮбҹҶбһ“бһ¶бһүбһҹбһ·бһҖбҹ’бһҹбһ¶бһҡбһ”бһҹбҹӢбһҹбһ¶бһҖбһӣбһңбһ·бһ‘бҹ’бһҷбһ¶бһӣбҹҗбһҷбһўбһ„бҹ’бһӮбһҡ бһ“бһ·бһ„бһҜбһҖбһҹбһ¶бһҡбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҖбҹҸбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҠбһҖбһҹбҹ’бһҡбһ„бҹӢбһ–бһёбһ”бҹ’бһҡбһ—бһ–бһҹбҹ’бһҡбһ”бһ…бҹ’бһ”бһ¶бһ”бҹӢ бһ“бһ·бһ„бһўбһ¶бһ…бһҷбҹ„бһ„бһ”бһ¶бһ“бһ•бһ„бһҠбҹӮбһҡбҹ”

### бҹЈ.бҹ§.бҹў. бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹҶбһ”бҹӮбһҖбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ

бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ¶бһ“бһҒбһ·бһҸбһҒбҹҶбһҡбһҖбҹ’бһҹбһ¶бһҖбһ¶бһҡбһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҠбҹӮбһӣбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһңбһ·бһ—бһ¶бһӮбҹ” бһӣбһҫбһҹбһ–бһёбһ“бҹҒбҹҮ бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһ·бһҸ (real data), бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҖбҹ’бһӣбҹӮбһ„бһҸбһ¶бһҳ persona (team-simulated data) бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ·бһ”бҹ’бһ”бһ“бһ·бһҳбҹ’бһҳбһ·бһҸ (synthetic data) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ”бҹӮбһҖбһұбҹ’бһҷбһ…бҹ’бһ”бһ¶бһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ—бһ¶бһ“бҹӢбһ…бҹ’бһҡбһЎбҹҶбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһҖбһҹбҹ’бһҡбһ¶бһҷбһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбҹ”

### бҹЈ.бҹ§.бҹЈ. бһҖбһ¶бһҡбһ’бһ¶бһ“бһ¶бһӮбһ»бһҺбһ—бһ¶бһ– бһ“бһ·бһ„бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһӣбһ‘бҹ’бһ’бһ•бһӣ

бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһ”бһ¶бһ“бһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһӣбһҫбһӮбһ»бһҺбһ—бһ¶бһ– бһ“бһ·бһ„бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһӣбһ‘бҹ’бһ’бһ•бһӣ бһҠбҹ„бһҷбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ” бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶ бһ“бһ·бһ„бһ•бҹ’бһ‘бҹҖбһ„бһ•бҹ’бһ‘бһ¶бһҸбҹӢбһұбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”бһҮбһ¶бһҳбһҪбһҷбһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһң бһҒбһҺбҹҲбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҖбҹӮбһӣбһҳбҹ’бһўбһҮбһ¶бһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһҖбҹҶбһ бһ»бһҹбһҠбҹӮбһӣбһўбһ¶бһ…бһҖбһҫбһҸбһҳбһ¶бһ“бҹ”

бһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһҖбһ¶бһҡбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҠбҹ„бһҷбһ—бһ¶бһ–бһҹбҹ’бһҳбҹ„бҹҮбһҸбҹ’бһҡбһ„бҹӢ бһҳбһ·бһ“бһ”бҹҶбһ•бҹ’бһӣбһҫбһҹ бһ“бһ·бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„бҹ” бһҖбҹ’бһҡбһ»бһҳбһҹбҹ’бһҡбһ¶бһңбһҮбҹ’бһҡбһ¶бһңбһҖбҹҸбһ”бһ¶бһ“бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ—бһ¶бһ–бһҹбһҳбһҹбҹ’бһҡбһ”бһҡбһңбһ¶бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһёбһҖбһҳбҹ’бһҡбһ„бһҹбҹҶбһҺбһҪбһҡ бһ“бһ·бһ„бһӣбһ‘бҹ’бһ’бһ•бһӣбһҺбҹӮбһ“бһ¶бҹҶбһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһҠбһҫбһҳбҹ’бһ”бһёбһ’бһ¶бһ“бһ¶бһҗбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһ”бһ‘бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҸбҹӮбһ„ бһ“бһ·бһ„бһўбһ¶бһ…бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ“бҹ’бһҗбҹӮбһҳбһ”бһ¶бһ“бһ“бһ¶бһ–бҹҒбһӣбһўбһ“бһ¶бһӮбһҸбҹ”
